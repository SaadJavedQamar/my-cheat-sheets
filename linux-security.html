Essential Linux Security Commands

Production-ready bash snippets for server hardening, SSH security, firewall configuration, and system auditing. Copy, paste, secure.

SSH Hardening

4 commands

Disable Root Login

Beginner

Prevent direct root access via SSH by modifying the SSH daemon configuration. Critical first step for any server hardening.

sudo sed -i 's/#PermitRootLogin yes/PermitRootLogin no/' /etc/ssh/sshd_config && sudo systemctl restart sshd
SSH Config Critical

Enforce Key-Based Auth

Intermediate

Disable password authentication completely. Only allow SSH key pairs for authentication.

sudo sed -i 's/#PasswordAuthentication yes/PasswordAuthentication no/' /etc/ssh/sshd_config && sudo systemctl restart sshd
SSH Keys 2FA

Change Default SSH Port

Beginner

Move SSH from default port 22 to reduce automated brute-force attacks. Remember to update firewall rules first.

sudo sed -i 's/#Port 22/Port 2222/' /etc/ssh/sshd_config && sudo systemctl restart sshd
SSH Port Obfuscation

Install Fail2Ban

Intermediate

Intrusion prevention framework that bans IPs after failed login attempts. Essential for internet-facing servers.

sudo apt update && sudo apt install fail2ban -y && sudo systemctl enable fail2ban --now
Fail2Ban IPS Automation

Firewall (UFW/iptables)

4 commands

Enable UFW with Default Deny

Beginner

Activate UFW firewall with default-deny policy. Blocks all incoming traffic except explicitly allowed.

sudo ufw default deny incoming && sudo ufw default allow outgoing && sudo ufw enable
UFW Firewall Default-Deny

Allow SSH with Rate Limiting

Intermediate

Allow SSH access with rate limiting (6 connections per 30 seconds) to prevent brute force attacks.

sudo ufw limit 22/tcp comment 'SSH with rate limit'
UFW Rate-Limit SSH

Save iptables Rules

Advanced

Make iptables rules persistent across reboots using iptables-persistent package.

sudo apt install iptables-persistent -y && sudo netfilter-persistent save
iptables Persistent Netfilter

Remove UFW Rule by Number

Beginner

List numbered rules and delete specific entries. Safer than deleting by pattern matching.

sudo ufw status numbered && sudo ufw delete [NUMBER]
UFW Cleanup Maintenance

System Auditing

4 commands

Check Last Logins

Beginner

Display last login times for all users. Essential for detecting unauthorized access.

lastlog | grep -v "Never"
Audit Login Monitoring

Find SUID/SGID Files

Advanced

Locate files with SUID/SGID bits set. Common privilege escalation vectors if misconfigured.

find / -perm /6000 -type f 2>/dev/null | grep -v "snap\|podman"
SUID Privilege-Esc Hunting

List Open Ports

Beginner

Display all listening ports with process information. Use `ss` (modern) instead of deprecated `netstat`.

sudo ss -tulnp | grep LISTEN
Network Ports Recon

Run Lynis Security Audit

Intermediate

Comprehensive security auditing tool for Unix-based systems. Checks 300+ security controls.

sudo apt install lynis -y && sudo lynis audit system --quick
Lynis Audit Compliance

File Permissions

4 commands

Find World-Writable Files

Intermediate

Locate files writable by anyone. Security risk if found in system directories.

find / -xdev -type f -perm -0002 -print 2>/dev/null
Permissions World-Writable Risk

Set Sticky Bit on /tmp

Beginner

Prevent users from deleting others' files in shared directories. Usually set by default on /tmp.

sudo chmod +t /tmp && ls -ld /tmp | grep -o 'rwt'
Sticky-Bit /tmp Shared

Secure /tmp Mount Options

Advanced

Mount /tmp with noexec, nosuid, nodev flags to prevent execution of malicious binaries.

sudo mount -o remount,noexec,nosuid,nodev /tmp && echo "/tmp defaults,noexec,nosuid,nodev 0 0" | sudo tee -a /etc/fstab
Mount noexec Hardening

Set Immutable Attribute

Advanced

Make critical files immutable (even root cannot modify without removing attribute). Use for /etc/passwd, binaries.

sudo chattr +i /etc/passwd && sudo lsattr /etc/passwd | grep -q "\-i\-" && echo "Protected"
chattr Immutable Advanced
